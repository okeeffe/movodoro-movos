# My Movement Snacks (Movos)

This repository contains my personal movement snacks for use with [movodoro](https://github.com/okeeffe/movodoro).

## Categories

- **Balance & Stillness** - Balance work and stillness practices
- **Carry & Crawl** - Loaded carries and crawling patterns
- **Club Flow** - Steel club movement flows
- **Get-Up Practice** - Turkish get-up variations
- **Micro Strength** - Quick strength work
- **OS Resets** - Original Strength resets and developmental positions
- **Reset & Breath** - Breathing and reset practices
- **Tribal Squats** - Squat variations
- **Tumbling & Ground Flow** - Ground-based movement flows

## Usage

Point movodoro to this directory:

```bash
export MOVODORO_MOVOS_DIR=~/coding/personal/movodoro-movos
```

Then use normally:

```bash
movodoro get
movodoro everyday
movodoro report
```

## Note

These are tailored to my equipment, training preferences, and RPE perception. Your RPE values may differ!
